# 🎌 Crunchyroll Content Analytics Framework 

> Anime streaming analytics framework focused on engagement intelligence, churn prediction, retention analytics, experimentation, and executive KPI reporting.

---

# 📊 Executive Overview

This project simulates how streaming platforms like Crunchyroll measure content performance across acquisition, engagement, retention, and subscriber behavior.

The framework models how analytics teams evaluate anime title performance, user journeys, discovery behavior, and churn risk to support executive-level content strategy decisions.

---

# 🚀 Features

✅ Synthetic anime streaming datasets  
✅ Viewer engagement analytics  
✅ Content performance KPI framework  
✅ Churn prediction modeling  
✅ Retention intelligence reporting  
✅ Discovery → Watch → Return attribution analysis  
✅ Experimentation analytics  
✅ Executive-style dashboard visuals  
✅ Streamlit dashboard integration

# 📈 KPIs Analyzed

| KPI | Description |
|---|---|
| Title Starts | Number of users starting a title |
| Episode Completion Rate | Viewer completion percentage |
| Watch Hours | Total engagement hours |
| Return Rate | Repeat session behavior |
| Churn Risk | Subscriber cancellation probability |
| Discovery Conversion | Click-to-watch effectiveness |
| Retention Lift | Impact of engagement on retention |
| Localization ROI | Performance by language region |
